A Simple Audio Playback Application

A custom, interactive audio player built using standard HTML, CSS, and JavaScript.

Features
- Audio Playback: Custom Play/Pause controls.
- Volume Control: Interactive slider for audio level management.
- Progress Tracking: Real-time playback time display with a seekable progress bar.
- Playback Speed**: Adjustable audio speed (0.5x to 2.0x).
- Mute Functionality: Quick toggle to mute/unmute.
- Keyboard Shortcuts: 
    - Press Space to play/pause.
    - Press M to mute/unmute.

Technologies Used
- HTML5
- CSS
- JavaScript
